URI(Uniform Resource Identifier)

리소스를 고유하게 식별하는 일반적인 개념. 
URL와 URN을 포함하며, 어떤 리소스든 식별하는 데 사용됨.


URL(Uniform Resource Locator)

URI의 하위 집합. 
실제 리소스가 위치한 경로나 주소를 지정한다.
웹에서 많이 사용되며, 프로토콜, 도메인명, 포트 등으로 리소스의 위치를 명시한다.
인터넷을 통해 리소스를 찾거나 검색하는데 사용된다.


URN(Uniform Resource Name)

리소스의 위치나 방법을 나타내지 않고 단순히 리소스의 이름을 식별한다.
ISBN(국제 표준 도서 번호)와 같은 고유하게 식별할 수 있는 것을 URN이라고 한다.
