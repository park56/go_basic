# go-swagger

go-swagger는 Go 언어로 작성된 RESTful API를 빌드하고 문서화하는 도구입니다. 이 도구를 사용하면 쉽게 API 설계를 할 수 있으며, 자동으로 API 문서를 생성할 수 있습니다. 또한, go-swagger는 API 서버를 실행하기 위한 코드 생성을 지원합니다.

go-swagger는 Swagger 2.0 (OpenAPI 2.0) 사양을 준수하여 API 설계를 작성하고 문서화합니다. 이를 통해 API의 구조와 기능에 대한 명확한 이해를 돕습니다.

go-swagger를 사용하면 API 코드와 API 문서가 항상 일치하도록 유지할 수 있습니다. 이를 통해 API 변경 사항을 쉽고 빠르게 문서화하고, API 사용자들이 항상 최신 정보를 제공받을 수 있도록 할 수 있습니다.

## go-swagger의 기능

- Swagger 2.0 (OpenAPI 2.0) 사양 준수
- API 설계 및 문서화
- 코드 생성
- API 서버 실행
- API 변경 사항의 쉬운 문서화

## go-swagger의 장점

- API 설계와 문서화를 한 도구에서 처리 가능
- 코드와 문서가 항상 일치하여 API 변경 사항을 빠르게 적용 가능
- Swagger 2.0 (OpenAPI 2.0) 사양 준수로 인한 API 이해의 용이성
- 코드 생성을 통한 개발 생산성 향상

## 결론

go-swagger는 Go 언어로 작성된 RESTful API를 빌드하고 문서화하는 도구로, API 설계와 문서화를 쉽게 할 수 있습니다. 또한, 코드 생성을 지원하여 API 서버를 쉽게 실행할 수 있습니다. API 변경 사항을 빠르게 적용할 수 있게 해주는 이 도구는 개발 생산성을 높일 수 있는 매우 유용한 도구입니다.

## 사용방법

swag init

swag i

변경사항이 있을 떄 마다 swag i
